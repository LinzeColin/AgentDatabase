# GSAP Skills

This is a reference-only wrapper for the official `greensock/gsap-skills` repository installed globally for Codex.

Use the specific official skills for execution:

- `gsap-core` for core tweens, easing, transforms, staggers, and matchMedia.
- `gsap-timeline` for sequencing, labels, nesting, and playback.
- `gsap-scrolltrigger` for scroll-linked animation, pinning, scrub, refresh, and cleanup.
- `gsap-plugins` for plugin registration, Flip, Draggable, SplitText, MorphSVG, DrawSVG, and plugin licensing.
- `gsap-utils` for `gsap.utils` helpers such as clamp, mapRange, random, snap, toArray, wrap, and pipe.
- `gsap-react` for React and Next.js patterns with `useGSAP`, refs, context, and cleanup.
- `gsap-frameworks` for Vue, Nuxt, Svelte, SvelteKit, and other lifecycle-based frameworks.
- `gsap-performance` for animation performance, transform-first motion, batching, and reducing jank.

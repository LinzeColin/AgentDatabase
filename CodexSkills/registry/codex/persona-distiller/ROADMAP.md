# Roadmap

Post-v0.0.0.4 candidates, not current requirements:

- pluggable semantic near-duplicate detector with a dependency-free fallback;
- interactive Claim review UI;
- permission-brokered connectors with revocation and retention controls;
- cross-host behavioral portability benchmark;
- richer multimodal source extraction with source-frame citation;
- signed package manifests and reproducible build attestation.

# Quarantined Hypothesis Builder

## Activation

Run only when `meta.json.existential_hypotheses_enabled` is true and the user explicitly requested interpretive depth.

## Objective

Explore possible existential stakes, obsessions, core tensions, legacy motives, or relationships to mortality without converting them into facts or diagnosis.

## Required entry

```markdown
## Hypothesis: ...
- Status: interpretive hypothesis, not fact
- Supporting Claim/source IDs:
- Independent contexts:
- Alternative explanations:
- Counterevidence:
- Confidence band:
- Time scope:
- Falsification/downgrade condition:
- Harm if wrong:
- Runtime influence: none | limited, with rationale
```

Reject hypotheses based only on style, a single anecdote, public mythology, demographic stereotype, or leaked/private data. Write exclusively to `hypotheses.md` and hypothesis Claim records.

# Examples

```bash
# Single identity
python3 scripts/init_target.py --name "Richard Feynman" --identity 1 --workspace ./workspaces

# Weighted multi identity
python3 scripts/init_target.py --name "Example Person" --identity "1:60+5:40" --workspace ./workspaces

# Optional scenario hint
python3 scripts/init_target.py --name "Example Founder" --identity 2 --scenario "公司战略与产品评审" --workspace ./workspaces
```

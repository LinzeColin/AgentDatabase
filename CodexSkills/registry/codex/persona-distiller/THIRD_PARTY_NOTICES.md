# Third-party notices and research influences

本项目的实现代码为本交付包原创/此前 Persona Distiller 代码的迭代，没有复制 Nuwa、God 或 dot-skill 的源码正文。设计受到下列公开项目和研究启发：

- `alchaincyf/nuwa-skill`：六路人物研究、资料落盘、已知/未知/声音测试、双 Agent 精炼。
- `fattly/god-skill`：价值、思想谱系、盲点、矛盾和隔离的存在性假设。
- `titanwings/colleague-skill` 的 `dot-skill` 分支：Work/Persona 分离、纠错、版本、回滚和私域材料生命周期。
- OpenPersona：身份/记忆/状态分离、影响边界、快照与回滚。
- Mind Clone / Mind-Cloning-Engineering：Claim 到来源的可追溯链接、可审计运行产物和价值权重。
- Generative Agents、CoALA、Reflexion、Letta：记忆、反思、计划、行动和长期状态的架构模式。
- OpenAI Agent Skills / skill-creator：渐进披露、入口最小化、脚本确定性、评测和安装约定。

各项目名称和商标归其权利人。本包不包含其数据、模型权重、受版权保护语料或真人背书。

# Read-only research synthesis

- Nuwa contributes six-lane research, file persistence, known/unknown/voice tests and dual-role refinement.
- God contributes cross-context mental models, value hierarchy, contradictions, intellectual lineage, blind spots and optional existential interpretation; the latter remains quarantined.
- dot-skill contributes Work/Persona separation, editable intermediate artifacts, corrections, snapshots and rollback.
- OpenPersona and stateful-agent systems reinforce separating identity constitution, mutable state, memory and influence boundaries.
- Generative Agents shows observation, memory retrieval, reflection and planning each contribute to believable behavior; CoALA provides modular memory/action/decision architecture; Reflexion motivates episodic feedback without weight updates.
- Evidence-grounded mind-clone projects reinforce canonical source links and run artifacts; SkillRL/Darwin-style systems motivate validation-gated extraction of successful procedures and failures.
- OpenAI Agent Skills guidance motivates progressive disclosure, deterministic scripts, explicit metadata and regression evaluation.

Project stars, demos and self-scores are adoption signals, not fidelity benchmarks. No public unified benchmark establishes a single best person-distillation system.

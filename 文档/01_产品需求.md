# 产品需求

## 一、这是什么

AgentDatabase 是所有 agent（Codex、Claude Code）历史数据的唯一备份源。
会话记录、token 用量、大型归档都存在这里，用 Release 附件承载超过仓库限制的大文件。

## 二、给谁用

| 用户 | 要什么 |
|---|---|
| 负责人 | 数据不丢，随时能取回，一份可信的备份 |
| Memory Atlas | 一个稳定的分析数据源 |

## 三、不做什么

- 不放业务代码，只放数据与其校验信息。
- 不对外公开敏感会话内容（视仓库可见性设置而定）。

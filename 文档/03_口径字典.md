# 口径字典

## 一、关键概念

| 项 | 裁定 | 状态 |
|---|---|---|
| 备份完整性 | 每个归档上传后必须 sha256 逐字节回传校验通过 | 已定 |
| 唯一备份源 | 本仓库是 agent 历史的最终备份，删除前必须确认此处有副本 | 已定 |

## 二、外部数据形态

| 来源 | 必须长什么样 | 状态 |
|---|---|---|
| 会话归档 | tar.gz 或加密分片，附 MANIFEST 与 sha256 | 已定 |

## 三、恒定为真的规则

| 规则 | 说明 |
|---|---|
| 校验后才删源 | 任何数据从别处删除前，本仓库副本必须已校验 |

## 四、证据等级定义

| 等级 | 含义 |
|---|---|
| `已提取` | 从实际归档核对过 |
| `已声明` | 只有记录说法 |

## 五、报告等级定义

| 等级 | 含义 |
|---|---|
| 合规 | 备份齐全且校验通过 |

## 六、术语对照

| 英文 | 中文 | 说明 |
|---|---|---|
| Owner | 负责人 | 决策与验收责任人 |
| Stage | 阶段 | 路线图一级单元 |
| Phase | 步骤 | 阶段下二级单元 |
| Task | 任务 | 步骤下三级单元 |
| Roadmap | 路线图 | 完整计划 |
| Release | 发布附件 | GitHub 承载大文件的方式 |
| Memory | 记忆 | Memory Atlas 的分析对象 |
| Atlas | 图谱 | Memory Atlas 产品名 |
| agent | 智能体 | Codex、Claude Code 等自动化开发助手 |

## 结论

S03 P2 固定一条边界：credential is not memory。

`credentials_not_transcript` 的意思是：普通 transcript 可以进入公开 raw，
但 cookie、session token、password、api key、private key、oauth token 和
browser credential store 不能进入 GitHub。

## 会被拦截的内容

- cookie。
- session token。
- password。
- api key。
- private key。
- oauth token。
- browser credential store。

## 不应该被拦截的内容

- 普通对话 transcript。
- 讨论 token budget、API key 管理原则、密码管理器概念的文字。
- 不包含真实凭证值的同步说明、需求说明和审计记录。

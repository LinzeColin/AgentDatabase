# airM2 macdata 功能清单

## 当前功能

1. 设备预检：仅允许目标 `airM2`，以本机 `MacBook Air / Apple M2 / 8GB` 事实为准。
2. 明文指标采集：硬件、电池、存储、内存、进程摘要、Git 状态、Docker/Homebrew 可用性、目录体积。
3. 隐私边界：不采集 Time Machine，不使用 iCloud，不读取 API key、token、password、cookie、session、Keychain、shell history、完整环境变量或 `.env` 原文。
4. GitHub 归档：每次运行 commit + push 到 `macdata-airM2`，并用远程 hash 校验。
5. 本机保留：远程验证成功后，只保留最近 3 天 `airM2` macdata 数据、报告、运行记录和 macdata 临时缓存。
6. macdata 自身清理：远程验证成功后，只清理本设备目录下 3 天以前数据、报告、运行记录和 `data/cache/archive_push` 临时缓存。
7. 中文报告：输出全中文报告，包含指标、上传状态、清理状态、ROI、风险、SWOT、失败项。

## 明确不做

- 不读取或合并其他设备目录。
- 不设置 launchd、cron、本地守护进程或登录项。
- 不自动清理 Docker、Homebrew、系统缓存、项目缓存。
- 不执行 `docker system prune`、`brew cleanup` 或删除 `~/Library/Caches`。
- 不删除项目 `node_modules`、`.venv`、`build`、`dist`、源码或业务数据。

# airM2 macdata 模型参数文件

## Codex Automation 参数

| 参数 | 当前值 | 说明 |
|---|---|---|
| codex_model | gpt-5.3-codex-spark | 自动化建议模型 |
| reasoning_effort | xhigh | 需要可靠预检、上传、清理门控 |
| schedule_cron | 10 1 * * * | Australia/Sydney 每天 01:10 |
| timezone | Australia/Sydney | 本机运行时区 |

## 设备参数

| 参数 | 当前值 | 说明 |
|---|---|---|
| device_key | airM2 | 设备唯一键 |
| expected_model_keywords | MacBook Air | 以本机事实为主 |
| expected_chip_keywords | M2 | 以本机 Apple M2 为准 |
| expected_memory_gb_min | 8 | 本机 8GB |
| retention_days | 3 | 本机仅保留最近 3 天 |

## 风险阈值

| 参数 | 当前值 | 说明 |
|---|---|---|
| storage_yellow_free_gb | 70 | 剩余空间低于该值黄色预警 |
| storage_red_free_gb | 40 | 剩余空间低于该值红色预警 |
| swap_yellow_gb | 1 | swap 达到该值黄色预警 |
| swap_red_gb | 4 | swap 达到该值红色预警 |

## 清理参数

| 参数 | 当前值 | 说明 |
|---|---|---|
| docker_cleanup | true | 执行 `docker system prune -f`，不含 volumes，不含 `-a` |
| homebrew_cleanup | true | 执行 `brew cleanup -s`，不使用 sudo |
| user_cache_cleanup | true | 只清理配置列出的 `~/Library/Caches/*` 子目录 |
| project_cache_cleanup | true | 只清理 `.pytest_cache`、`.mypy_cache`、`.ruff_cache`、`__pycache__`、`node_modules/.cache`、`.next/cache` |
| project_cache_max_deleted_paths | 200 | 单次最多删除 200 个项目缓存目录 |
